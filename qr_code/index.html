QR Image Analysis: Manual and QR modes

The analysis application has the ability to read QR codes embedded within images. This allows the user to:

  1. Add additional information associated to the analysis within the QR code.
  2. Automate the image analysis with minimal human intervention.

The following sections cover how the above can be achieved.

Manual mode

Apple

Steps for Manual analysis using QR code

  1. Once the application has initialized, the user must set the SOURCE TYPE as MANUAL.
  2. Next the crop type option must be set to APPLE .
  3. The analysis option must be set to ANALYZE RED AREA .

  4. The image file containing the overlayed QR code that maintains additional information related to the analysis must be selected. This will ensure that the additional data will be available as part of the report.

  5. Based on the arrangement of the fruit specimens in the image, the user must set the number of rows and columns in the image in the ROWS and COLUMNS field.

  6. The SPECIMEN COUNT field must be set to a value that reflects the number of specimens in the supplied image that must be analyzed.
  7. The user must next select an RGB/PDF file that contains values to train the model.
  8. If the user requires detailed images of every step of the analysis, the user must set the GET DETAILED RESULTS option to YES, otherwise it must be set to NO.
  9. Once all the options have been set and reviewed, the user must click the PROCESS button once.
  10. Once the process has initialized, the progress can be tracked using the progress bar.
  11. Once completed, the DOWNLOAD button will be enabled. Clicking the button will initiate the download of a zip file from the default browser in the user's system.
  12. Once the zip file has been downloaded, the user can extract the files and analyze the images and data therein.
  13. If another analysis needs to be run, the button REFRESH can be clicked, which will clear all existing values on-screen and will allow the user to run another analysis.

QR Code mode

Apple

Steps for Automatic analysis

  1. Once the application has initialized, the user must set the SOURCE TYPE as QRCODE. This will cause many of the input fields to remain disabled.

  2. The image file containing the overlayed QR code that maintains all the required details related to the analysis must be selected.

  3. Once the inputs have been set and reviewed, the user must click the PROCESS button once.

  4. Once the process has initialized, the progress can be tracked using the progress bar.
  5. Once completed, the DOWNLOAD button will be enabled. Clicking the button will initiate the download of a zip file from the default browser in the user's system.
  6. Once the zip file has been downloaded, the user can extract the files and analyze the images and data therein.
  7. If another analysis needs to be run, the REFRESH button can be clicked, which will clear all existing values on-screen and will allow the user to run another analysis.
